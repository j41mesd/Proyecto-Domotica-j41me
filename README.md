# j41mesd.github.io
Código utilizando Arduino y ScadaBR, para controlar los sensores de una vivienda.

# Abstract
Español:
Este proyecto consiste en la domotización de una vivienda, que en este caso va a ser un plano de una vivienda en un folio, y que va a estar puesto en un tablero de madera. Los componentes que se usaran seran: Sensores de temperatura, sensores de iluminación, led, un "motor" y la placa Arduino.En el Arduino, definiremos todas las variables que se van a usar durante el codigo de Arduino, para que funcione todos los sensores que tiene y usando las librerias de scada, que la necesitaremos instalar para que funcione los componentes en ScadaBR. Una vez instalado, usaremos el ScadaBR para utilizar los sensores y que se vean en una visualización previa que tiene la aplicación para que podamos ver y controlar los sensores.

Ingles:
This project involves automating a home, which in this case will be a house plan drawn on a sheet of paper and mounted on a wooden board. The components used will be: temperature sensors, light sensors, LEDs, a motor, and an Arduino board. In the Arduino, we will define all the variables used in the Arduino code to ensure the sensors function correctly. We will also use the SCADA libraries, which we will need to install to enable the components to work in ScadaBR. Once installed, we will use ScadaBR to access the sensors and view them in a preview within the application, allowing us to see and control them.
